# DoStaffkin

Веб-приложение для управления доставками. Позволяет пользователям создавать заказы доставки, рассчитывать стоимость маршрута и отслеживать статус отправлений.

## Функциональность

- **Главная страница** — информационный раздел и навигация
- **Создание заказа** — форма с указанием адреса отправки/доставки, размера груза, скорости доставки, контактных данных
- **Отслеживание** — поиск статуса отправления по номеру заказа
- **Интеграция с картами** — Яндекс.Карты для расчета маршрутов

## Технологический стек

- **Frontend:** Angular 21.1.1, TypeScript 5.9
- **Стили:** CSS
- **HTTP клиент:** HttpClient (Angular)
- **Формы:** Reactive Forms (Angular)
- **UI уведомления:** ngx-toastr 20.0.4
- **Асинхронность:** RxJS 7.8.2
- **Сборка:** Angular CLI 21.1.1
- **Тестирование:** Vitest 4.0.18
- **API:** REST (testologia.ru/delivery)

## Установка и запуск

### Требования
- Node.js 18+
- npm 10.9.2

### Команды

```bash
# Установка зависимостей
npm install

# Локальная разработка (http://localhost:4200)
npm start

# Сборка для production
npm build

# Watch режим (автоматическая пересборка)
npm run watch

# Тесты
npm test
```

## API эндпоинты

- `POST /delivery/create` — создание новой доставки
- `GET /delivery/info` — получение информации о доставке (параметр: `id`)

## Маршруты

- `/` — главная страница
- `/order` — создание заказа
- `/track` — отслеживание
